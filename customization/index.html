    Klasifikasi dengan KNN (K-Nearest Neighbors) menggunakan python¶

    1. Persiapan¶

    Sebelum membuat program, siapkan dahulu Datasets dan Packages nya.

    • Datasets bisa di download di link dibawah ini :

    https://archive.ics.uci.edu/ml/datasets/Bank+Marketing

    • Packages yang dibutuhkan
    • Python 3.7
    • Spyder 3.3.2
    • Numpy
    • Pandas
    • Scikit Learn

    2. Import Package¶

    Setelah semua package yang dibutuhkan sudah di download dan sudah terinstall dalam library Python, Import Package ke Code editor Spyder dengan memasukan code dibawah ini :

    import pandas as pd
import numpy as np

    3. Import dataset¶

    Setelah dataset terdownload, import datasets dengan code dibawah ini :

    deposit = pd.read_csv("bank.csv",delimiter =";")
pd.set_option('display.max_columns', 19)
print(deposit.head(),'\n')

    • fungsi .read_csv() digunakan untuk membaca file csv

    • fungsi .head() digunakan untuk menampilkan data 6 teratas

    3. Mengubah Data Katagorikal menjadi Data Numerik¶

    Dataset yang digunakan masih terdapat data katagorikal, langkah selanjutnya mengubah data katagorikan menjadi data numerik menggunakan code berikut :

    #Mengubah Variabel menjadi Data Kategorik
deposit_new = deposit.select_dtypes(include=['object']).copy()
nRows, nCols = deposit_new.shape
for myIndex in range(0,nCols):
    headerName = deposit_new.columns[myIndex]
    deposit_new[headerName] = deposit_new[headerName].astype("category")
    deposit_new[headerName] = deposit_new[headerName].cat.codes
    deposit[headerName] = deposit_new[headerName]
deposit_numeric = deposit
print( "data transform \n",deposit_numeric.head(),"\n")

    4. Memilih Variabel¶

    Untuk menamai Variabel independen diberi nama x. Untuk mempermudah maka keluarkan variabel y (dependen) dari dataset yang berarti bahwa semua variabel kecuali variabel y merupakan variabel independen

    #variabel independen
x = deposit_numeric.drop(["y"], axis = 1)
x.head()

#variabel dependen
y = deposit_numeric["y"]
y.head()

    5. Klasifikasi KNN¶

    Untuk menggunakan package KNN, terlebih dahulu harus memangil package Scikit learn terlebh dahulu, dengan menggunakan code

    from sklearn.neighbors import KNeighborsClassifier

    Kemudian panggil fungsi KNN dari package Scikit Learn dengan code 

    knn = KNeighborsClassifier(n_neighbors=5)

    6. Spliting Data Train dan Data Test¶

    Untuk melakukan pembagian data train dan test, pertama perlu mengimport fungsi untuk membagi data dari library scikit-learn. Scikit-Learn memberikan sejumlah fitur untuk keperluan data science.

    Dataset yang ada akan dibagi menjadi 2 bagian yaitu x test dan x train untuk variabel independen, y test dan y train untuk variabel dependen.

    #SPLIT DATA
from sklearn.model_selection import train_test_split
x_train, x_test, y_train, y_test = train_test_split(x, y, test_size = 0.2, random_state = 123)

    7.Scaling Data¶

    Digunakan untuk menyamakan dan men standartkan data dengan menggunkan fungsi StandarScaler dari dalam package Scikit Learn proses ini juga digunakan untuk membandingkan data dengan unit yang berbeda dengan cara memasukan code dibawah ini :

    #Scale data
from sklearn import preprocessing
scaler = preprocessing.StandardScaler()
scaler.fit(x_train)

x_train = scaler.transform(x_train)  
x_test = scaler.transform(x_test)

    8. Train dan Testing data¶

    Train data yang sudah di standarkan dari dengan code berikut :

    knn.fit(x_train, y_train)

    setelah data di train, selanjutnya data di test dengan code berikut :

    #hasil prediksi
y_pred = knn.predict(x_test)
print('\ndata prediksi \n',y_pred,'\n')

    9. Membuat Confusion Matrix¶

    Confusion matrix adalah suatu metode yang biasanya digunakan untuk melakukan perhitungan akurasi

    import fungsi confusion matrix dari package scikit learn dengan code berikut:

    from sklearn.metrics import classification_report, confusion_matrix
print("prediksi =",confusion_matrix(y_test, y_pred))

    Output :

    prediksi = [[788  20]
 [ 76  21]]

    Berdasarkan output dari confusion matrix diketahui bahwa dengan metode K-Nearest Neighbor :

    • Jumlah prediksi seseorang memiliki klien berlangganan deposito berjangka (0) dan data aktual sesuai atau benar jika memiliki (0) adalah sebanyak 788.
    • Jumlah prediksi seseorang memiliki klien berlangganan deposito berjangka (0) dan data aktual tidak sesuai atau seseorang tidak memiliki klien berlangganan deposito berjangka (1) adalah sebanyak 76.
    • Jumlah prediksi seseorang tidak memiliki klien berlangganan deposito berjangka (1) dan data seseorang memiliki klien berlangganan deposito berjangka (0) adalah sebanyak 20.
    • Jumlah prediksi seseorang tidak memiliki klien berlangganan deposito berjangka (1) dan seseorang tidak memiliki klien berlangganan deposito berjangka (1) adalah sebanyak 21.
