    Implementasi Pohon Keputusan Menggunakan Python¶

    1. Persiapan¶

    Download dataset dahulu menggunakan link dibawah ini :

    disini

    Selanjutnya siapkan package yang diperlukan dibawah ini :

    • Python 3.7
    • Spyder 3.3.2
    • Numpy
    • Pandas
    • Scikit Learn
    • graphviz
    • pydotplus

    2. Import package yang diperlukan¶

    Setelah semua package yang dibutuhlan telah terdownload, import ke dalam Spyder seperti code dibawah ini :

    import pandas as pd
from sklearn.tree import DecisionTreeClassifier 
from sklearn.model_selection import train_test_split
from sklearn import metrics 
from sklearn.externals.six import StringIO  
from IPython.display import Image  
from sklearn.tree import export_graphviz
import pydotplus

    3. Load data¶

    Load data yang sudah didownload tadi, usahakan satu folder dengan file yang akan dibuat ini:

    col_names = ['pregnant', 'glucose', 'bp', 'skin', 'insulin', 'bmi', 'pedigree', 'age', 'label']
# load dataset
pima = pd.read_csv("diabetes1,csv", header=None, names=col_names)

    4. Seleksi Fitur¶

    Dalam step ini, perlu pembagian fitur. Untuk variabel X digunakan untuk fitur dan variabel Y digunakan untuk target.

    feature_cols = ['pregnant', 'insulin', 'bmi', 'age','glucose','bp','pedigree']
X = pima[feature_cols] # fitur
y = pima.label # target

    5. Split data¶

    Selanjutnya split data menjadi 2 untuk data test dan data train

    #split data : 30% data test, 70% data train
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=1)

    6. Membuat Model Pohon keputusan¶

    Gunakan package Scikit-learn untuk permodelan pohon keputusan

    # untuk klasifikasi objek
clf = DecisionTreeClassifier(criterion="entropy", max_depth=3)

# untuk train pohon keputusan
clf = clf.fit(X_train,y_train)

#untuk prediksi data yg diuji
y_pred = clf.predict(X_test)

#menampilkan akurasi
print("Accuracy:",metrics.accuracy_score(y_test, y_pred))

    output akurasi

    Accuracy: 0.7705627705627706

    7. Visualisasi Pohon Keputusan¶

    Visualisasi data menggunakan code berikut 

    dot_data = StringIO()
export_graphviz(clf, out_file=dot_data,  
                filled=True, rounded=True,
                special_characters=True, feature_names = feature_cols,class_names=['0','1'])
graph = pydotplus.graph_from_dot_data(dot_data.getvalue())  
graph.write_png('diabetes.png')
Image(graph.create_png())
